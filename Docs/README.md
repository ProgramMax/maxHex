## Support

* Compilers:
    * GCC*
    * Clang*
    * Visual C++
* OSes:
    * Some POSIX Linux (can't test them all)*
    * Windows
    * MacOS*
* Instruction sets:
    * x86 (64)
    * ARM*

*Coming soon

If you would like max to support more contacts us and let us know.
